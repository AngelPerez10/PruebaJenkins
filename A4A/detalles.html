Justificacion

Nombre de usuario

Esto crea una etiqueta de texto que indica al usuario qué tipo de información se espera en el campo de entrada siguiente, que se refiere al nombre de usuario.

Contraseña

Aquí se crea un campo de entrada de tipo "password", lo que significa que los caracteres ingresados se ocultarán para proteger la privacidad del usuario.

Repetir Contraseña

Aquí se crea nuevamente un campo de entrada de tipo "password", para proteger mas al usuario

Direccion

Aquí se crea un campo de entrada de texto donde el usuario puede escribir su dirección

Ciudad

Aquí se crea un campo de entrada de texto donde el usuario puede escribir su ciudad donde vive

Provincia

Aquí se crea un campo de entrada de texto donde el usuario puede escribir su provincia

Numero de Huespedes

Servicios

Se proporciona una lista de checkboxes que permiten a los usuarios seleccionar diferentes servicios. Cada checkbox tiene un nombre (name) que se utilizará para identificar el servicio cuando se envíe el formulario, y un valor (value) asociado a ese servicio. Los servicios incluyen: Aire Acondicionado Almohadas a la carta Caja Fuerte Detector de humos Minibar Mesa de trabajo Conexión de internet Wifi

Tipo de Habitacion

Se proporcionan opciones para que los usuarios seleccionen el tipo de habitación deseado. Cada opción tiene un radiobutton (Input type="radio") con un nombre (name) y un valor (value): Individual Doble Suite

Pension

Se proporcionan opciones para que los usuarios seleccionen el plan de pensión deseado: Alojamiento Desayuno Media pensión Pensión completa

Comentario

Crea un área de texto donde los usuarios pueden escribir comentarios o sugerencias. El atributo name se utilizará para identificar este campo cuando se envíe el formulario.

Email

Crea un campo de entrada de tipo "email" donde los usuarios pueden ingresar su dirección de correo electrónico.
